# Itinerary Assistant - 智能旅行规划助手

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Itinerary Assistant** 是一个现代化的智能旅行规划应用，旨在通过AI技术简化和增强您的旅行体验。只需通过语音或文字描述您的旅行想法，即可获得一份包含详细行程、预算估算和地图路线的个性化旅行计划。

## ✨ 功能特性

- **AI 智能规划**：基于强大的大语言模型根据您的目的地、时间和偏好等需求自动生成详细的旅行规划。
- **语音输入**：支持通过语音快速创建行程或记录开销，解放您的双手。
- **预算与开销管理**：在生成行程时自动估算预算，并提供简单易用的记账功能与预算分析功能，让您随时掌握开销情况和预算使用情况；
- **交互式地图**：在地图上直观地展示每日行程活动点位和路线，方便您规划出行。
- **云端同步**：所有行程数据都安全地存储在云端，支持多设备随时访问。

## 🚀 快速开始

1.  **安装 Docker**:
    确保您的系统已安装 [Docker](https://www.docker.com/) 和 [Docker Compose](https://docs.docker.com/compose/)。

2.  **下载发布包**:
    访问项目的 [Releases 页面](https://github.com/tinglansli/ItineraryAssistant/releases)，下载最新的部署包。

3. **配置API KEY**：
    解压下载的文件，然后编辑`.env.example`文件，在其中配置必要的`api-key`，然后另存为`.env`文件。

3.  **启动服务**:
    根据您的操作系统执行相应的启动脚本：
    - **Windows**: 双击 `start.cmd`。
    - **macOS/Linux**: 在终端中执行 `start.sh`。

4.  **访问应用**:
    打开浏览器并访问 `http://localhost:80`。


## 📖 使用指南

欢迎使用 Itinerary Assistant！以下是主要的使用流程：

1.  **注册与登录**
    - 首次使用时，您需要注册一个账户。
    - 登录后，您将进入应用主页。

2.  **创建新行程**
    - 在主页上，点击“创建行程”按钮。
    - 您可以通过**文字**输入目的地、日期和旅行偏好。
    - 或者，使用**语音输入**功能，直接说出您的旅行需求。
    - 您可以点击右上角的“编辑偏好”按钮，查看和更新您的默认旅行偏好。
    - 提交后，AI 将为您生成一份详细的行程预览。

3.  **查看行程详情与确认**
    - 创建成功后，您将看到包含每日活动、地图路线和预算估算的行程详情。
    - 如果对预览的行程满意，点击“确认行程”，系统会将其永久保存在您的账户下。

4.  **管理您的行程**
    - 在主页上，点击“我的行程”按钮。
    - 在“我的行程”页面，您可以查看所有已确认的行程列表。
    - 点击任意行程即可进入详情页进行管理。

5.  **预算分析与开销记录**
    - 在行程详情页，您可以：
        - 进入**预算分析**页面，查看各项预算与实际开销的对比。
        - 进入**记录开销**页面，通过文字或语音快速记录旅途中的每一笔花费。

## 🏛️ 项目架构

项目采用前后端分离的设计，并通过 Docker Compose 进行容器化部署。

- **前端 (Frontend)**：基于 `Vue.js` 和 `Vue Router` 构建的单页应用 (SPA)，负责用户界面和交互。
- **后端 (Backend)**：基于 `Spring Boot` 的 RESTful API 服务，负责处理业务逻辑、数据存储和与第三方服务集成。
- **数据库 (Database)**：使用 `Google Firestore` 作为主数据库，存储用户、行程和开销等核心数据。
- **缓存 (Cache)**：使用 `Redis` 缓存临时的行程预览和热点数据，提高响应速度。
- **第三方服务**:
  - **大语言模型 (LLM)**: 集成**阿里云百炼**大模型，用于生成行程计划。
  - **语音转文字 (STT)**: 集成**科大讯飞**语音服务，用于处理语音输入。
  - **地图服务**: 使用**高德地图 API**，用于地理位置查询和地图展示。
